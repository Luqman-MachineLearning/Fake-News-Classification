
# 📰 Fake News Classifier

A machine learning project that classifies news articles as **real or fake** using natural language processing (NLP) techniques.

This project focuses on building an end-to-end ML pipeline — from data preprocessing to model training and evaluation.

---

## 🚀 Overview

Fake news detection is a critical problem in today's digital world. This project applies machine learning to automatically identify misleading or false information in news content.

---

## 🧠 Features

- Text preprocessing and cleaning
- Feature extraction using NLP techniques
- Model training and evaluation
- Binary classification (Real vs Fake news)
- Performance metrics (accuracy, precision, recall)
